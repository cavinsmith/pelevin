## 1. Мистические концепции

### Буддизм
- **Определение:** Философская основа творчества Пелевина — учение о пустоте, сознании и освобождении
- **Ключевые произведения:** «Чапаев и Пустота», «t», «Бэтман Аполло», «Смотритель», «Тайные виды», «Непобедимое Солнце»
- **Wiki-страницы:**
  - [[concepts/buddhizm]] — основная статья
  - [[mystical/pustota]] — шуньята
  - [[analysis/filosofskie-ssylki#1-буддизм]] — философские отсылки
  - [[analysis/religioznye-ssylki#1-буддизм]] — религиозные отсылки
- **Связи:** [[concepts/pustota]], [[concepts/karma]], [[concepts/mistika]], [[concepts/shamanizm]]
- **Эволюция:** 1996 (основа) → 1999-2009 (критика) → 2018-2022 (исцеление)

### Шаманизм
- **Определение:** Древний метод познания через изменённые состояния сознания, прямое общение с духами
- **Ключевые произведения:** «Числа», «Священная книга оборотня», «Бубен верхнего мира», «Нижняя тундра»
- **Wiki-страницы:**
  - [[concepts/shamanizm]] — основная статья
  - [[mystical/shamanizm]] — мистический анализ
  - [[analysis/filosofskie-ssylki#2-даосизм]] — даосизм
  - [[analysis/religioznye-ssylki#4-шаманизм]] — религиозные отсылки
- **Связи:** [[concepts/buddhizm]], [[concepts/okkultizm]], [[concepts/mistika]], [[concepts/magiya-slova]]
- **Эволюция:** 1990 (антропология) → 1993-1998 (практика) → 2002 (личный опыт)

### Оккультизм
- **Определение:** Тайные знания и практики, магия, каббала, масонство
- **Ключевые произведения:** «Лампа Мафусаила», «Священная книга оборотня»
- **Wiki-страницы:**
  - [[concepts/okkultizm]] — основная статья
  - [[mystical/magiya-slova]] — магия слова
- **Связи:** [[concepts/shamanizm]], [[concepts/mistika]], [[concepts/magiya-slova]]

### Мистика
- **Определение:** Опыт трансцендентного, непосредственное переживание абсолютного
- **Ключевые произведения:** «Чапаев и Пустота», «Бэтман Аполло», «Непобедимое Солнце»
- **Wiki-страницы:**
  - [[concepts/mistika]] — основная статья
  - [[mystical/SYSTEM]] — синтез мистической системы
- **Связи:** [[concepts/buddhizm]], [[concepts/shamanizm]], [[concepts/okkultizm]]

### Пустота (шуньята)
- **Определение:** Фундаментальная природа реальности — отсутствие независимого существования
- **Ключевые произведения:** «Чапаев и Пустота», «t», «Бэтман Аполло»
- **Wiki-страницы:**
  - [[concepts/pustota]] — основная статья
  - [[mystical/pustota]] — мистический анализ
- **Связи:** [[concepts/buddhizm]], [[concepts/realnost-illuziya]], [[concepts/identichnost]]

### Карма

- **Ключевые произведения:** Рассказы о шаманизме, «Числа»
- **Wiki-страницы:**

## 6. Связи между концепциями

### Мистические связи

```
Буддизм → Пустота → Сознание → Свобода
    ↓
Шаманизм → Оккультизм → Мистика
    ↓
Карма → Любовь → Солнце
```

### Технологические связи

```
Киберпанк → Трансгуманизм → ИИ
    ↓
Медиа → Манипуляция → Контроль
```

### Философские связи

```
Философия → Идентичность → Реальность/Иллюзия
    ↓
Свобода → Язык как реальность → Власть и контроль
```

### Социальные связи

```
Социум → Магия слова → Язык как реальность
    ↓
Медиа → Бизнес → Любовь → Семья → Смех
```

### Кросс-системные связи

```
Буддизм ↔ Трансгуманизм (оба — трансформация сознания)
Шаманизм ↔ Киберпанк (оба — изменённое состояние)
Пустота ↔ Реальность/Иллюзия (оба — онтология)
Карма ↔ Власть (оба — причинность)
```

## 7. Эволюция концепций

### Хронология появления

```
1996: Буддизм как философская основа
    ↓
1999-2009: Буддизм как инструмент критики
    ↓

```

```
2011: Трансгуманизм как science fiction
    ↓

    ↓

```
