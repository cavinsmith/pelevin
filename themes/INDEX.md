## Центральная ось

Все темы связаны через три фундаментальных концепции (из [SYSTEM.md](../mystical/SYSTEM.md)):

1. **Пустота (шуньята)** — основа всего. Все темы возвращаются к пустоте.
2. **Сознание** — создаёт реальность. Все темы связаны с восприятием.
3. **Слово (нарратив)** — инструмент создания реальности. Все темы связаны с языком.

## Как использовать этот анализ

### Для написания нового произведения
1. Выберите доминирующую тему (или их сочетание).
2. Определите период, в стиле которого хотите писать.
3. Изучите эволюцию темы в указанном периоде.
4. Используйте цитаты как ориентир тона.
5. Следите за парадоксами — они ключ к стилю Пелевина.

### Для глубокого анализа тем
1. Начните с базового файла темы (например, `seksualnost.md`).
2. Перейдите к эволюционному файлу (например, `seksualnost-2.md`).
3. Изучите промежуточные файлы (например, `seksualnost-3.md`, `seksualnost-4.md`).
4. Прочитайте финальный файл трансформации (например, `smert-7.md`, `vlast-7.md`, `identichnost-6.md`, `realnost-6.md`, `svoboda-4.md`, `seksualnost-6.md`, `konspiraciya-6.md`, `biznes-7.md`, `religiya-6.md`, `politika-6.md`).
5. Сравните эволюцию: от ранних проявлений к финальному тождеству.
6. Обратите внимание на парадоксы и противоречия — они ключ к глубине.

### Для анализа нового произведения
1. Определите, какие темы присутствуют.
2. Сравните с эволюцией тем в известных произведениях.
3. Проверьте, есть ли новые повороты в развитии темы.
